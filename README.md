# 北航网站相关油猴脚本


## Accoding - BUAA Accoding Add Copy Button

在Accoding平台上添加代码和输入输出复制按钮。

[accoding.js](accoding.js)


## SPOC-VIDEO - BUAA SPOC Video Speed Hack

在SPOC平台上实现视频倍速播放和防止暂停功能，挂《国家安全》视频。

[spoc-video.js](spoc-video.js)


## We Learn - We Learn Patch

在We Learn平台上实现自动答题和视频倍速播放功能。
食用方法：点击Welearn课程学习页面上方的标题栏。

[we-learn.js](we-learn.js)



## BUAA JUDGE HELPER - BUAA Judge Fast Copy

在BUAA JUDGE计算机作业平台上复制试题。
食用方法：点击题目序号。

[buaa-judge-helper.js](buaa-judge-helper.js)


## BUAA DMPlatform - BUAA DMPlatform Helper

在BUAA DMPlatform（离散数学教学平台）上查找答案。
食用方法：点击题目序号。

[buaa-dmplatform.js](buaa-dmplatform.js)
