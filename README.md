# 北航网站相关油猴脚本

## BUAA-SPOC-Helper

针对智学北航平台的增强插件，支持 MSA 回放字幕/PPT 导出、课程详情视频/PPTX 下载，以及 SPOC 播放器辅助工具，刷《国家安全》

[buaa-spoc.helper.js](buaa-spoc-helper.js)

[GitHub仓库](https://github.com/peter-erer/buaa-spoc-helper)

## Accoding - BUAA Accoding Add Copy Button

在Accoding平台上添加代码和输入输出复制按钮。

[accoding.js](accoding.js)


## SPOC-VIDEO - BUAA SPOC Video Speed Hack

在SPOC平台上实现视频倍速播放和防止暂停功能，挂《国家安全》视频。

[spoc-video.js](spoc-video.js)


## We Learn - We Learn Patch

在We Learn平台上实现自动答题和视频倍速播放功能。
食用方法：点击Welearn课程学习页面上方的标题栏。

[we-learn.js](we-learn.js)



## BUAA JUDGE HELPER - BUAA Judge Fast Copy

在BUAA JUDGE计算机作业平台上复制试题。
食用方法：点击题目序号。

[buaa-judge-helper.js](buaa-judge-helper.js)


## BUAA DMPlatform - BUAA DMPlatform Helper

在BUAA DMPlatform（离散数学教学平台）上查找答案。
食用方法：点击题目序号。

[buaa-dmplatform.js](buaa-dmplatform.js)


## MSA-Download-Video - BUAA MSA Classroom Video Downloader

在BUAA MSA Classroom平台上下载课程视频。
食用方法：进入课程页面 (https://classroom.msa.buaa.edu.cn/coursedetail?course_id=*) 后点击下载按钮。一般一节课会对应两个视频（教师和屏幕）。另外，支持通过浏览器新标签页预览视频，以及下载本节课的PPT（纯图片版）。脚本的下载功能有一点bug，在批量下载时可能触发失败/卡顿。

[msa-download-video.js](msa-download-video.js)
